Welcome to Python GDS with Nazca’s documentation!

To isntall the library please clone the git repo https://github.com/MartinMiroslavovMihaylov/Python_GDS.git to a empty folder. The you need to use the .rar file from the lib to install python nazca lib since thi script is just an raparound the nazca lib.

How to use the script

After installing the Scrip you can use the Controll.py script where you can see how the single GDS sctructures are made. And in the secound path there is an for loop Example that will generate an GDSII file with all the objects available in the library.
